## üèõ Framework Architecture Patterns

### üåü **Conversation-Based** (AutoGen, CAMEL)
Agents communicate through structured dialogues and message passing

### üìä **Graph-Based** (LangGraph, MetaGPT)
Agents organized in directed graphs with state transitions

### üëî **Role-Based** (CrewAI, ChatDev) 
Agents assigned specific roles and responsibilities in team structures

### üîÑ **Pipeline-Based** (LangChain, LlamaIndex)
Sequential processing with modular components and tool chains

### üéØ **Task-Oriented** (BabyAGI, SuperAGI)
Goal-driven agents that break down objectives into executable tasks

## üöÄ Getting Started

### Quick Setup Examples

#### **CrewAI - Multi-Agent Team**
```python
from crewai import Agent, Task, Crew

# Define agents
researcher = Agent(
    role='Researcher',
    goal='Research comprehensive information on AI agents',
    backstory='Expert researcher with deep knowledge of AI'
)

writer = Agent(
    role='Writer', 
    goal='Write engaging blog posts about technology',
    backstory='Skilled technical writer with flair for storytelling'
)

# Define tasks
research_task = Task(
    description='Research latest trends in AI agents',
    agent=researcher
)

write_task = Task(
    description='Write a blog post based on research',
    agent=writer
)

# Create crew
crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, write_task],
    verbose=True
)

# Execute
result = crew.kickoff()
```

#### **LangChain - Tool-Using Agent**
```python
from langchain.agents import initialize_agent, Tool
from langchain.llms import OpenAI

# Define tools
tools = [
    Tool(
        name="Search",
        func=search_function,
        description="Search the web for information"
    )
]

# Create agent
llm = OpenAI(temperature=0)
agent = initialize_agent(
    tools=tools, 
    llm=llm, 
    agent="zero-shot-react-description"
)

# Run agent
result = agent.run("Research the latest AI agent frameworks")
```

We welcome contributions to improve this curated list! Please follow these guidelines:

### How to Contribute
1. **Fork** this repository
2. **Create** a new branch (`git checkout -b add-framework`)
3. **Add** your framework following the format below
4. **Ensure** the framework meets our criteria:
   - Active maintenance (commits in last 6-12 months)
   - Significant community adoption (>500 stars OR backing from known organizations)
   - Well-documented and production-ready
   - Unique value proposition

### Submission Format
```markdown
| Framework Name | [GitHub Link](https://github.com/...) | Stars | Brief description (2-3 lines) | [Paper/Blog Link](https://...) | Tags |
```

### Quality Standards
- **No duplicate entries** or forks of existing frameworks
- **No experimental repos** without significant traction
- **No commercial-only** solutions (open-source preferred)
- **Clear documentation** and examples required
